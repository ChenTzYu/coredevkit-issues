# CoreDevKit Feedback & Issues

Welcome! If you found a bug, have a feature request, or just want to suggest an improvement for [CoreDevKit](https://coredevkit.com), please [open an issue](../../issues).

### Privacy & GDPR Requests
For sensitive matters such as data deletion or privacy inquiries, please do not open a public issue. 
Instead, you can reach out via email: `privacy[at]coredevkit [dot] com` (replace `[at]` and `[dot]`).
